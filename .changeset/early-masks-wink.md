**Box, atoms:** Add `stretch` value to `alignItems`

Extends the `alignItems` options to include `stretch`.

**EXAMPLE USAGE:**
```jsx
<Box alignItems="stretch" />
```

```jsx
import { atoms } from 'braid-design-system/css';

atoms({ alignItems: 'stretch' });
```
