Add optional `tooltipPlacement` prop to `ButtonIcon`

The `tooltipPlacement` prop allows you to specify the placement of the tooltip to either `top` or `bottom`.
The default value is `top`.

**EXAMPLE USAGE:**

```jsx
<ButtonIcon
  tooltipPlacement="bottom"
/>
```
