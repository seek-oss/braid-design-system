**Autosuggest, TextField:** Add `clearLabel` prop

To support translations, the clear button in the field can now be customised using the `clearLabel` prop.

**EXAMPLE USAGE:**
```jsx
<Autosuggest
  clearLabel="Clear"
  // ...
/>
```
