**Text, Heading:** Add icon slots

Provides a designed slot for adding an icon to `Text` and `Heading` components

**EXAMPLE USAGE:**

```jsx
<Text icon={<IconPromote />}>{...}</Text>
```

or with a `Heading`:

```jsx
<Heading level="3" icon={<IconPromote />}>{...}</Heading>
```
