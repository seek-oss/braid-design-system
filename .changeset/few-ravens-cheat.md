**ButtonIcon:** Add `small` size

Introduce a new `small` size for `ButtonIcon` component.
This size sits alongside the existing `standard` and `large` sizes.

**EXAMPLE USAGE:**
```jsx
<ButtonIcon
  size="small"
  icon={<IconEdit />}
  label="Small size"
/>
```
