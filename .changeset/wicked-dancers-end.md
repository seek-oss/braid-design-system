**MenuRenderer:** Add `small` size.

Introduce a new `small` size for the `MenuRenderer` component.
This is available via the `size` prop, which supports the existing `standard` (default) and `small`.

**EXAMPLE USAGE:**
```jsx
<MenuRenderer size="small" ... />
```
