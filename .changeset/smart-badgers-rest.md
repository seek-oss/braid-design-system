**Box, atoms:** Add `grid` value to `display`

Extends the `display` options to include `grid`.

**EXAMPLE USAGE:**
```jsx
<Box display="grid" />
```

```jsx
import { atoms } from 'braid-design-system/css';

atoms({ display: 'grid' });
```
